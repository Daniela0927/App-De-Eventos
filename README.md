 Descripción de la Aplicación

App de Eventos es una aplicación Android diseñada para permitir a los usuarios crear, visualizar y gestionar eventos importantes como reuniones, cumpleaños, actividades académicas y recordatorios.

La aplicación permite:
	•	Crear nuevos eventos con título, fecha, descripción y categoría.
	•	Guardar los eventos en una base de datos local.
	•	Listar los eventos creados.
	•	Editar y eliminar eventos.
	•	Interfaz simple y fácil de usar.
